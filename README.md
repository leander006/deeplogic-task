# Welcome user
- So repo is divided into 2 part
- Frontend in react js
- Backend in node js and express js
# Initial setup
  # Frontend
  - cd frontend 
  - ` npm install `
  - ` npm run dev `
  # Backend
  - cd backend
  - ` npm install`
  - ` npm run dev`

## .env for backend
